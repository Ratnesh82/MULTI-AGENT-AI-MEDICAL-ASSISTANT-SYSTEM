# MULTI-AGENT-AI-MEDICAL-ASSISTANT-SYSTEM
• Built a 10-agent AI system for automated clinical diagnosis and prescription recommendation, achieving 87.3% diagnostic accuracy across 10 common conditions. 
• Integrated LLM-based diagnosis using Google Gemini 2.5 Flash with rule-based drug interaction
checking via RxNorm and FDA APIs, implementing a 5-layer safety validation pipeline.
• Implemented mandatory physician-in-the-loop approval mechanism ensuring patient safety while
reducing diagnostic workload by an estimated 40-60% for routine primary care cases.
